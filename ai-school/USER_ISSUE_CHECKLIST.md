# ユーザー発行チェック（管理者）

## 1. 個別発行
1. 受講履歴管理 -> ユーザーID / パスワード発行
2. 氏名 / メール / 部署 / パスワード / ロール を入力
3. 個別発行
4. 下の一覧に追加されたことを確認

## 2. CSV一括発行
1. `templates/users-import-template.csv` を開いて必要行を編集
2. CSV内容を管理画面のテキストエリアへ貼り付け
3. CSV一括発行
4. 成功件数 / 失敗件数メッセージを確認

## 3. パスワード再発行
1. 一覧で対象ユーザーの `PW再発行`
2. 新パスワードを入力
3. 対象ユーザーでログインできることを確認

## 4. 有効/無効切替
1. 一覧で対象ユーザーを `無効化`
2. 対象ユーザーでログイン不可になることを確認
3. `有効化` でログイン可能へ戻ることを確認

## 5. ガード確認
- 最後の有効な管理者は無効化できない
- ログイン中の管理者自身は無効化できない

## 6. ユーザー削除
1. 一覧で対象ユーザーの `削除`
2. 確認ダイアログで削除を確定
3. 履歴削除の要否を選択（OK: 履歴も削除 / キャンセル: アカウントのみ削除）
